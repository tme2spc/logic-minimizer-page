Google Play 출시 준비 중

소개

Logic Minimizer는 카르노맵(Karnaugh Map)을 이용해 논리식을 최소화하고, 타이밍 다이어그램·회로도·HDL 코드까지 한 번에 생성해주는 디지털 논리회로 도구입니다. 디지털 논리회로를 배우는 학생부터 회로 설계를 다루는 엔지니어까지 활용할 수 있습니다.

주요 기능

카르노맵 최소화

카르노맵을 이용한 논리식 최소화 및 보강항(Don't Care) 처리

Timing Diagram

논리 회로의 타이밍 다이어그램 자동 생성

회로도 생성

최소화된 논리식으로부터 Schematic 회로도 생성

D Flip-Flop

D Flip-Flop 기반 순차 회로 기능 지원

VHDL / Verilog 코드 생성

VHDL 및 Verilog HDL 코드 자동 생성

Testbench 생성

VHDL / Verilog Testbench 자동 생성

파이프라인

파이프라인 구조 설계 지원
